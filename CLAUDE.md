# CLAUDE CODE 工作指南
此文件为 Claude Code 在次项目中工作时提供指导。
## 1.项目概览
新疆南疆旅游路书
## 2. 核心要求
1、定时间：每天开车/游玩时长上限，不要太累 
2、路线，理清环线/顺向线，必去大点排好，小众京东准备好. 
3、做好住宿推荐，尽量在交通枢纽/市中心/景区门口(200~300rmb) 
4、我们计划自驾，规划化好自驾的省钱和省心 
5、吃喝。列好本地特色和必吃老店，避开网红溢价店，预算下吃饭标准，不暴饮不委屈自己 
6、全程花费的所有预算 
7.全程的避坑提醒。 

## 3.展示要求
1、制作成网页，后续我会通过github托管静态网页，在手机端微信也能查看
2、要分层次，多维度的记录（比如总的路书概览，各个地方的规划，各个景点的规划，各个避坑的点等）

## 4.我的初步旅游计划
1、落地喀什后，落地当天再加一天在喀什玩
2、喀什->红山谷->奥依塔克冰川->白沙湖（在这住一晚）
3、白沙湖->喀拉库勒湖->塔合曼湿地->塔县（在这住一晚）
4、塔县->托格伦夏->红其拉甫->盘龙古道->班迪尔蓝湖->塔县
5、塔县->慕士塔塔冰川->木吉火山->喀什
## 5.工作流程

- **功能开发**： 我同意后你可以按照思路来开发。如果不同意，请继续询问我的意见。
